Create a New Vault

This page creates a new 2/3-multisig smart contract using 3 Trezors.

You and your co-signers will need a total of 3 Trezor hardware wallets. Not all 3 Trezors need to be co-located. Co-signers can use this application on their own computers and send you their addresses via a (preferably secure) side-channel such as email.

Each address corresponds to some BIP32 path on a co-signer's Trezor. Co-signers can select whichever BIP32 path they want or each use the default below. It's OK to re-use BIP32 paths from the same Trezor across different smart contracts. Regardless of what you choose to do, remember which Trezor and BIP32 path you used for each co-signer's address!
